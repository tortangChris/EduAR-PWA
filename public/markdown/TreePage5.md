# Tree Traversals

- **Traversal:** Visiting all nodes in a tree.
- **Depth-First Traversals:**
  - **Preorder (Root → Left → Right)**
  - **Inorder (Left → Root → Right)**
  - **Postorder (Left → Right → Root)**
- **Breadth-First Traversal (Level Order):** Visit nodes level by level.
