# Doubly Linked List

- **Definition:** Each node has **two pointers**:
  - **Prev** → previous node.
  - **Next** → next node.
- Traversal can be **forward** or **backward**.
- **Operations:** Easier deletion/insertion at both ends.
- **Example:**
  ```
  NULL ← [10|Prev,Next] ⇄ [20|Prev,Next] ⇄ [30|Prev,Next] → NULL
  ```
